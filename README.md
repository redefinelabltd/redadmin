# REDADMIN

A simple admin panel made by REDEFINELAB LTD (http://www.redefinelab.com)

## INSTALLATION INSTRUCTION

### NPM

```bash
npm install redadmin --save-dev
```

#### SETUP

#### javascript

```javascript
require('redadmin');
```

#### css

```sass
@import '~redadmin/dist/css/app';
```
